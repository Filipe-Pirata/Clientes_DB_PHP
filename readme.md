# Sistema de adminstração de usuarios em PHP

Neste repositório você encontrará um sistema para administração de usuarios desenvolvido em PHP com as seguintes funcionalidades:

<ul>
<li>Cadastro </li> 
<li>Login mediante validação de usuario e senha </li> 
<li>Listagem dos usuarios </li> 
<li>Edição dos usuarios cadastrados no BD. </li> 
<li>Exclusão dos usuarios.tro </li> 
<li>Sessões de acesso. </li> 
</ul>
    
Foram utilizadas neste projeto as seguintes tecnologias:  HTML, CSS, Bootstrap e PHP.
